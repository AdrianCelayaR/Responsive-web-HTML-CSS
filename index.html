Mis programas

E-commerce

Un e-commerce es una plataforma en línea donde se venden productos y/o servicios, permitiendo al usuario hacer compras cómodamente desde casa.

Cotizador

Un cotizador web es una herramienta en línea que permite al usuario obtener precios y presupuestos de productos o servicios de manera rápida y sencilla, a través de un formulario de solicitud en el sitio web de la empresa.

Responsive HTML & CSS
